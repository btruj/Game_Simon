Press A Key to Start

Instructions

A game where the player must repeat a sequence of button presses that are displayed by the computer.

  1. The game starts by pressing any key on the keyboard

  2. The player will see the text "Level 0" and the computer will start the first round by displaying a sequence of button presses

  3. The player must then repeat the sequence by clicking on the corresponding buttons in the same order

  4. Each time the player successfully repeats the sequence, the level increases and the sequence becomes longer.

If the player makes a mistake, the player can then start a new game by pressing any key on the keyboard.
